To run this code:
Install slim program (can only get slimGUI if you have a mac): https://messerlab.org/slim/
Unzip test_files.zip to your desktop: this contains the test input files. 
You can open the slim_splatche.slim file in the GUI and press the play button to run,
or use the command line.
To run on the command line, the basic command is:
$: slim slim_splatche.slim
For more run options:
$: slim
